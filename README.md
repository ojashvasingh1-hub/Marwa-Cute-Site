# Marwa Cute Site

A tiny, playful, non-confessional webpage made as a random cute surprise.

## Replace the media
- Put your video at `assets/video.mp4`
- Put your audio at `assets/local.mp3`

## Sync the page
Open `js/script.js`.
- Edit the `moments` array timings and original on-screen messages.
- Change `audio.currentTime = 0` if your chosen clip begins later.
- Change the `47` second final-card trigger to match your clip.

Open `index.html` in a browser to test.
